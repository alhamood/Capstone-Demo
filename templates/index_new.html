New York bike share redistribution engine:

Capstone Project for Data Incubator Fellowship.

Created by Albert W. Hamood, January - February 2016

This app uses a predictive model of user behavior in the Citibike bike-sharing system to recommend an efficient redistribution of bikes among stations that meets user demand. It was created from an analysis of 22+ million rides using Citibike's publicly available dataset.

The primary functionality, displayed geographically, is available at the first link below.

Below this are further links, containing more detailed information about the project including: the form of the model, and how I chose and cross validated it; the algorithm behind the recommendation of a redistribution strategy; and the motivation for the project, supported by some analysis. You can also check the performance of the model against observed data for any past date in the dataset I collected.
