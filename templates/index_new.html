New York bike share redistribution engine:

Capstone Project for Data Incubator Fellowship.

Created by Albert W. Hamood, January - February 2016

Behind this website is a predictive model using factors such as hour, month, day of the week, and weather, along with station-specific factors. Using this model, I predict the change in bikes over a user-defined period of time. I then recommend stations as sinks and sources for an efficient redistribution of bikes among stations to meet upcoming demand. These data are then visually displayed using CartoDB's API.

You can also check predictions of the model for a previous date to view accuracy. This model is based on an extremely random forest, although for predictions greater than 24 hours into the future, a decision tree is used which produces faster results.
